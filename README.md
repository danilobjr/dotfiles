# dotfiles
My dotfiles

## Installation

1. Clone repo and navigate to the directory

    ```git clone https://github.com/danilobjr/dotfiles.git && cd dotfiles```

2. Run 

    ```source ./install-dotfiles-cli.sh```

3. Now you can use dotfiles CLI. Try this

    ```dotfiles --help```
